## 什么是住宅IP归属？一句话先说清楚

**住宅IP归属**，指的是某个IP地址由哪家互联网服务提供商（ISP）分配、归属在哪个真实的住宅地址上。跟机房IP最大的区别就在这里：住宅IP的WHOIS记录指向真实的居民宽带用户，而机房IP则归属于数据中心运营商。

判断一个IP是不是真住宅，本质上就是看它的ISP归属。

## 住宅IP归属查询的3种主要方法

### 方法一：用在线IP信息查询工具

这是最直接的方式。输入IP地址，看返回结果里的几个关键字段：

1. **ISP（互联网服务提供商）**：显示AT&T、Comcast、Verizon这类运营商名称，基本是住宅IP；显示"Amazon"、"Alibaba Cloud"、"某某托管科技"，就是机房IP。
2. **Type（类型）**：真正的住宅IP会显示"Residential"；机房IP显示"Hosting"或"Datacenter"。
3. **Organization（组织）**：住宅IP通常显示运营商名称，或"Private Customer"这类字样。

常用的查询工具（纯文字提示，不附链接）：
- **ipinfo.io**：由前Facebook工程师创建，每天处理超200亿次查询，数据相对权威，ISP字段分辨率高
- **IPQualityScore**：除了归属信息，还提供IP风险评分，对跨境业务有参考价值
- **ipleak.net**：适合快速验证当前使用的代理IP是否有DNS泄漏问题
- **ARIN WHOIS**（美国互联网号码注册局）：最原始的归属数据来源，直接查 IP 段归属记录

### 方法二：看ASN和反向DNS

ASN（自治系统编号）是另一个判断指标。住宅ISP（如AT&T）有自己独立的ASN，和数据中心服务商的ASN完全不同。通过bgp.he.net可以查询某个IP段归属哪个ASN，再判断这个ASN的性质。

反向DNS（rDNS）也有参考价值。AT&T分配给住宅用户的IP，反向解析记录通常带有"res"、"dsl"、"comcast.net"等字样，格式接近 `174-123-45-67.res.bhn.net`。机房IP的rDNS记录则往往指向数据中心域名或干脆没有记录。

### 方法三：IP风险评分工具

IPQualityScore（IPQS）和Scamalytics这类工具，会综合多个维度给IP打分——是否在黑名单里、历史上是否被用于欺诈、是否是代理或VPN出口等。

真正的住宅IP，在IPQS上的评分通常在90分以上，黑名单记录为零。

这个方法尤其适合跨境金融、广告投放场景：你可以在使用IP之前先跑一遍风险评分，提前排掉有历史污点的IP。

## 住宅IP归属查询最容易踩的两个坑

**坑一：只看"Residential"标签，不看ISP名称**

有些服务商会在数据中心IP上打"Residential"标记，蒙混过关。真正的判断标准是ISP字段——它必须是真实的电信运营商，不能是某个科技公司。ISP显示"XYZ Technology Ltd"，无论type字段怎么写，都不是真住宅。

网上有些所谓的IP查询工具，查之前要你扫码登录或者填手机号。这种一律跳过。正规查询工具不需要你的个人信息，直接输IP出结果就行。
